Download and install the OSCAR Ground Station desktop application for Windows.

Windows x64 · ZIP Archive · v1.0.0

↓   OSCAR Ground Station — Version 1.0.0

Installation Instructions

  1. Click the download button above to save the ZIP file.
  2. Navigate to your Downloads folder and extract the ZIP file.
  3. Open the extracted folder and navigate to the x64 directory.
  4. Double-click OSCAR Ground Station-1.0.0 Setup.
  5. Follow the on-screen prompts to complete the installation.
